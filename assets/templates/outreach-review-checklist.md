# Template: Outreach Review Checklist (Module 07)

> The drafts are a starting line. Run each one through this before it goes out. The human send stays human.

---

## Per-message review

For each drafted message:

- [ ] The opening hook is specific to this person, not generic.
- [ ] Every fact in the message is true and came from their row. No invented details.
- [ ] It sounds like me, not like a robot or a different recruiter.
- [ ] It is short. I cut filler rather than adding.
- [ ] No "I hope this finds you well" or similar filler opening.
- [ ] It ends with one low-pressure question.
- [ ] Right channel for this candidate (LinkedIn vs email).
- [ ] Sending at a sensible time.

## The hard rule

- The agent drafts. A human reviews and sends. Always.
- The workflow may create a Gmail draft, never an auto-send.
- No message goes to a candidate without your eyes on it first.

## Quick quality bar

If you would be slightly embarrassed to receive it, do not send it. If it
reads like it could only have been written for this one person, send it.
