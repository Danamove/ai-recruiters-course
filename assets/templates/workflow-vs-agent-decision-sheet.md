## The framework, in three questions

1. Can I write the exact steps in advance, and they never change? -> **Workflow**
2. Does it need a judgment, or does the input vary a lot? -> add an **AI step**
3. Must the AI choose between several tools or actions on its own? -> only then a **full agent** (rare)

**Default: a workflow with one AI step. Reach for a full agent almost never.**
